# Project description
Ziggy is the web application that handle a calendar per cat. Place your chips on each day to manage things to do for your kitty !

## Owner-side
As an owner, add your precious cats, manage their names, weight, breed colors etc. each cat will automatically have a calendar ! Add and manage their vetenary appointment in it as feeding, change litter or buy some thing ! Manage the chips you are able to add on calendar and their color too. 
As an owner, you are able to add some person (pet sitter or friends nevermind) to a cat. this person can have access ot calendar to put some chips (chips allowed by owner only !).
They are invited by owner. 


# Admin 
Owner are register by the admin (admin is created via a one shot beginning command on install project). admin can see owner, cat pet sitter and manage them.
